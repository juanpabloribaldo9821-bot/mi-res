# Ribaldo Gourmet | Avant-Garde Gastronomy Festival

A sophisticated and modern landing page designed for a high-end culinary festival. This project focuses on luxury aesthetics, clear information hierarchy, and a seamless user experience across all devices.

## Description
This website serves as the official portal for the *Ribaldo Gourmet Festival*. It provides attendees with essential information about the concept, the executive chefs, the official schedule, and a visual gallery of the gastronomic experiences.

The design uses a "Dark Mode" palette with gold accents to evoke a sense of elegance and exclusivity, typical of signature cuisine environments.

## Technologies Used
* *HTML5:* Used for semantic web structure (main, section, article, header, footer).
* *CSS3:* For advanced styling, including:
    * *Flexbox & CSS Grid:* To create flexible and organized layouts.
    * *Media Queries:* To ensure full responsiveness on mobile, tablet, and desktop.
    * *Keyframe Animations:* For subtle fade-in effects and interactive hover states.
* *Google Fonts:* "Playfair Display" for elegant titles and "Poppins" for readable body text.

## Project Structure
The project is organized into a clean, modular structure:
* index.html: Contains the semantic structure and content of the site in English.
* style.css: Contains all visual styles, responsive design logic, and animations.
* logo1.png: Branding asset (ensure this file is in the root directory).
* README.md: Project documentation and setup instructions.

## Key Features
* *Responsive Design:* Optimized for mobile and desktop (>1024px) using dedicated media queries.
* *Semantic Data Table:* A structured schedule using <thead> and <tbody> tags for professional data display.
* *Interactive Gallery:* A grid-based image gallery with zoom and grayscale-to-color hover effects.
* *Smooth Navigation:* Internal links that allow users to jump between sections smoothly.
* *Executive Chef Section:* An additional section highlighting the talent behind the event.

## Setup Instructions
To view this project locally on your machine:
1.  *Download* the repository or the files (index.html, style.css, and logo1.png).
2.  *Organize:* Ensure all files are located in the same folder.
3.  *Launch:* Right-click on index.html and select *"Open with"* followed by your preferred web browser (Chrome, Firefox, or Edge).
4.  *Explore:* Scroll through the sections or use the navigation menu to test the responsiveness.
